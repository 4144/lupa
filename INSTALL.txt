Installing lupy
================

#) Download and unpack lupy

#) Download LuaJIT2

   http://luajit.org/download.html

#) Unpack the archive into the lupy base directory, e.g.::

     .../lupy-0.1/LuaJIT-2.0.0-beta4

#) Build LuaJIT::

     cd LuaJIT-2.0.0-beta4
     make
     cd ..

   If you need specific C compiler flags, pass them to ``make`` as follows::

     make CFLAGS="..."

#) Build lupy::

   python setup.py build
